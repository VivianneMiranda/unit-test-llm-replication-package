## Collect metrics

From the repository root:

```powershell
.\scripts\03_collect_metrics.ps1 -Project bcel -Origin developer
.\scripts\03_collect_metrics.ps1 -Project bcel -Origin opus-4.5
```

Paths are resolved via [`config/artifact-locations.json`](../config/artifact-locations.json).
